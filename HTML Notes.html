Document Structure Tags

<!DOCTYPE>, <html>, <head>, <body>
<!DOCTYPE>: Declares document HTML version. <!DOCTYPE html> for HTML5.
<html>: Root of document. Attributes: lang, dir.
<head>: Metadata container like title, links, scripts.
<body>: Visible body content of page.
Attributes for <html>:
  • lang: document language code
  • dir: text direction (ltr, rtl, auto)
Attributes for <body>:
  • onload, global attributes
<!DOCTYPE html>
<html lang="en" dir="ltr">
  <head>
    <title>Sample Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>Sample paragraph</p>
  </body>
</html>

Hello World

Links and Media Tags

<a>, <img>, <picture>, <map>, <area>
<a>: Hyperlink. Attributes: href, target, rel, etc.

Forms and Input Tags

<form>, <fieldset>, <legend>
<label>, <input>, <textarea>, <button>
<select>, <option>, <optgroup>, <datalist>
<form>: Container for inputs to submit data.
<fieldset>: Groups form controls.
<legend>: Caption for <fieldset>.

Table Tags

<table>, <caption>,
<thead>, <tbody>, <tfoot>,
<tr>, <th>, <td>,
<colgroup>, <col>
Structure tags for tables:
- <table>: Table container.
- <caption>: Table title/description.
- <thead>: Table header group.
- <tbody>: Table body group.
- <tfoot>: Table footer group.
- <tr>: Table row.
- <th>: Header cell.
- <td>: Data cell.
- <colgroup>: Group of columns.
- <col>: Column properties.
Attributes examples:
  • colspan, rowspan for <th>, <td>
  • scope for <th>
  • span for <colgroup> and <col>
  • border on <table> (deprecated but used here for demo)

Embedded Content Tags

<iframe>, <embed>, <object>, <param>, <canvas>, <svg>
<iframe>: Embed another webpage inside current page.
<embed>: Embed external resource like PDF or multimedia.
<object>: Embeds external resource, with fallback content.
<param>: Parameters for <object>.
<canvas>: Drawable area controlled by JavaScript.
<svg>: Scalable vector graphics container.
Attributes vary widely per tag, see above for details.
<iframe src="https://example.com" width="300" height="200" title="Example iframe"></iframe>

<embed src="test.pdf" type="application/pdf" width="300" height="200">

<object data="movie.swf" type="application/x-shockwave-flash" width="300" height="200">
  <param name="quality" value="high">
  Alternate content
</object>

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" />

Scripting Tags

<script>, <noscript>
<script>: Client-side scripting. External or inline JavaScript.
<noscript>: Alternative content if scripts disabled.
  • src: External file URL
  • async, defer: Loading behavior
  • type, nomodule
<script>
  console.log('Hello from script');
</script>

<noscript>
  <p>Please enable JavaScript.</p>
</noscript>
Console log: "Hello from script" (no visual output)
If JavaScript disabled, shows: Please enable JavaScript.

Deprecated / Obsolete Tags

<font>, <center>, <big>, <strike>, <applet>, <basefont>, <dir>, <frame>, <frameset>, <acronym>, <tt>
These tags are deprecated in HTML5; do not use. Prefer CSS and modern alternatives.
Examples:
<font> — use CSS font-family, color
<center> — use CSS text-align:center;
<big> — use CSS font-size
<strike> — use <del> or <s>
<applet> — use <embed> or <object>
<basefont> — use CSS
<dir> — use <ul>
<frame>, <frameset> — use CSS layouts
<acronym> — use <abbr>
<tt> — use CSS font-family: monospace;
